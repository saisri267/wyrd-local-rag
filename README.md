# AI Engineer - Trial By Fire (RAG System)

A professional-grade, local-first RAG (Retrieval-Augmented Generation) application designed for high-performance document analysis and grounded AI interactions.

## 🚀 Features

-   **Zero-Cloud Privacy**: Everything runs locally using Ollama and local vector storage.
-   **Strict Grounding**: The system is engineered to answer only based on the provided PDF context, explicitly stating when information is missing.
-   **Page-Aware Retrieval**: Every answer includes detailed citations with page numbers and document source.
-   **Modern Architecture**: 
    -   **Backend**: FastAPI, LangChain, ChromaDB, HuggingFace Transformers.
    -   **Frontend**: React (TS), Tailwind CSS, Framer Motion, Lucide icons.
-   **Engineering Transparency**: Dedicated pages explaining technical decisions and automated support workflows.

## 🛠 Prerequisites

-   **Python 3.10+**
-   **Node.js 18+**
-   **Ollama**: [Download Ollama here](https://ollama.com/), install it, and ensure the service is running.
